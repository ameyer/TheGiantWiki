Release Date:

Savage Posse is a hand-finished screenprint that was retailed through the 1996 Alternate Graphics catalog. It originally retailed for $5 plus $1 for shipping. The original name listed for this print was "Savage Giants."

Run size information is believed to represent the combined total for Savage Posse and the variant Savage Posse (Holly Solomon Gallery Edition). This design was later used in the 2006 print Giant Savage 2.
