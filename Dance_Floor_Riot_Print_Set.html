Year: 2011
Run Size: 150
Size: 12x12
Paper: unknown
Print Type: Screen Print

The 36 prints in this set were available exclusively as part of a box set, which includes the aforementioned matching-number prints, a stencil, a sticker sheet, a CD, and a certificate of authenticity. The box itself is embossed and has gold leaf detailing. A portion of the run (50 sets) was available for in-person purchase at the opening of Revolutions: The Album Cover Art of Shepard Fairey, at Robert Berman Gallery C2, on 03/12/2011. Additional sets were sold via Obeygiant.com on 06/28/2011, although it is unclear whether the remainder of the sets were released, or merely just another portion. The set originally retailed for $950.
