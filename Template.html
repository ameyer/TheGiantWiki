The following templates were created exclusively for use on TheGiant.Org:

{{Amazon}}

This amazon template is used to promote dontations to TheGiant.Org.

{{Copyright}}

This template was created to signify the legal rights of applied images within an article.

{{Gallery}}

This template was created to be the content backbone for individual fine art articles.

{{Inuse}}

This template is designed to denote pages that are withgoing major edits.

{{iTunes}}

This template generates iTunes links for specific bands. The artist number is required for this template to work. This number can be retrieved by right-clicking on the artist "Biography," selecting "Copy iTunes Music Store URL," and pasting the URL somewhere where it can be viewed. The number is after "artistId=".

{{Print}}

This template was created to be the content backbone for individual print articles.

{{PrintSet}}

This template is used exclusively for print sets.

{{Run Size|Number}}

This template was created to be used as the content for the Run Size Categories.

{{Size}}

This template was created to be used as the content for the print Size Categories.

{{Sticker}}

This template was created to be the content backbone for individual Sticker articles.

{{Stub}}

Use this template to acknowledge articles that need attention.

{{Year}}

This template was created to be used as the content for the Year Categories.
