Revolutionary Women Print Set

Year: 2005
Run Size: 50
Size: 30x42
Paper: unknown
Print Type: Screen Print

Large Format. Two of the four were later released as offset lithographs: Muslim Woman Offset and Revolution Woman Offset. Smaller versions of the prints were even later released as part of special Supply and Demand Book box set celebrating 20 years of the Obey Giant campaign, which was exclusively available at ICA Boston: Angela Davis (ICA Edition), Muslim Woman (ICA Edition), Revolution Woman (ICA Edition), and Zapatista Woman (ICA Edition).

Prints in This Set: Angela Davis - Muslim Woman - Revolution Woman - Zapatista Woman
