Roxy Music Print Set

Year: 2006
Run Size: 40
Size: 22x29
Paper: unknown
Print Type: Screen Print

Signed by both Steve Jones and Shepard Fairey, these prints were available only as a complete, six print set for $500. These images were based on Karl Stoecker's 1972 work on Roxy Music's self-titled debut album.

Prints in This Set: Roxy Music Andy Mackay - Roxy Music Brian Eno - Roxy Music Bryan Ferry - Roxy Music Graham Simpson - Roxy Music Paul Thompson - Roxy Music Phil Manzanera
